## 🐛 常见问题

### Q1: 看不到粒子背景？
**A**: 
- 检查浏览器版本（需Chrome 90+）
- 查看控制台是否有JS错误
- 确认 particles-tech.js 已加载

### Q2: 粒子太多/太少？
**A**: 
```javascript
// 修改 index.html 初始化代码
particleCount: 150  // 增加到150个
particleCount: 50   // 减少到50个
```

### Q3: 卡顿/性能差？
**A**:
- 减少粒子数量
- 禁用发光效果: `glowEffect: false`
- 关闭部分动画效果

### Q4: 移动端效果太多？
**A**: 
已自动优化，如需进一步简化：
```css
/* 禁用所有动画 */
* { animation: none !important; }
```
