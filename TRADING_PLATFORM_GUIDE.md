# 🎮 模拟交易平台使用指南

## 📋 功能概述

ICSFP 模拟交易平台是一个**虚拟股票交易系统**，让用户可以：
- 💰 使用虚拟资金进行股票交易
- 📈 实时查看投资组合收益
- 📊 跟踪交易历史记录
- 🏆 参与收益排行榜竞争
- 🎯 练习交易策略，零风险学习

---

## 🚀 快速开始

### 1. 访问交易平台
```
http://127.0.0.1:5000/trading
```

### 2. 初始设置
- **初始资金**: $100,000（虚拟美元）
- **交易股票**: 33只主流股票
- **手续费**: 0.1% (买入和卖出都收取)
- **交易时间**: 7x24小时（模拟系统）

### 3. 开始交易
1. 在"交易面板"选择股票
2. 查看当前价格
3. 输入交易数量
4. 点击"买入"或"卖出"

## 💡 功能详解

### 📊 账户统计

#### 可用资金
- 当前可用于购买股票的现金
- 买入股票会减少，卖出股票会增加

#### 总资产
- 现金 + 所有持仓的市值
- 实时更新，反映投资组合价值

#### 收益率
- `收益率 = (总资产 - 初始资金) / 初始资金 × 100%`
- 绿色为盈利，红色为亏损

#### 交易次数
- 买入和卖出的总次数
- 体现交易活跃度

---

### 💱 交易面板

#### 股票选择
- 支持 33 只主流股票
- 包括科技、金融、能源、医药等行业

#### 实时价格
- 基于真实股价的模拟价格
- 每次查询会有 ±2% 的随机波动
- 模拟市场动态变化

#### 交易数量
- 必须为正整数
- 买入时需要足够资金
- 卖出时需要足够持仓

#### 预计金额
- 自动计算交易总额
- 包含 0.1% 的手续费
- 买入示例：
  ```
  股价: $100
  数量: 10
  小计: $1,000
  手续费: $1 (0.1%)
  总计: $1,001
  ```

---

### 📦 持仓管理

#### 持仓列表
显示所有持有的股票：
- **股票代码**: AAPL, MSFT, etc.
- **股数**: 持有数量
- **成本**: 平均买入价格
- **现价**: 当前市场价格
- **盈亏**: 未实现盈亏和百分比

#### 成本计算
```
平均成本 = (已持股数 × 旧均价 + 新买入股数 × 新价格) / 总股数
```

**示例**：
```
第一次买入: 10股@$100 → 平均成本 $100
第二次买入: 5股@$110 → 平均成本 = (10×100 + 5×110) / 15 = $103.33
```

---

### 📜 交易记录

#### 记录内容
- **时间**: 交易发生时间
- **类型**: BUY (买入) / SELL (卖出)
- **股票**: 交易的股票代码
- **数量**: 交易股数
- **价格**: 成交价格
- **金额**: 交易总金额
- **盈亏**: 卖出时显示实际盈亏

#### 盈亏计算（卖出时）
```
盈亏 = (卖出价格 - 平均成本) × 股数
盈亏率 = (卖出价格 - 平均成本) / 平均成本 × 100%
```

**示例**：
```
买入: 10股@$100（成本 $1,000）
卖出: 10股@$120
盈亏: ($120 - $100) × 10 = $200
盈亏率: ($120 - $100) / $100 = 20%
```

---

### 🏆 排行榜

#### 排名规则
- 按**总收益率**排序（不是绝对金额）
- 前三名显示奖牌 🥇🥈🥉
- 最多显示前 10 名

#### 评分指标
- **总资产**: 当前总价值
- **收益率**: 投资回报率
- **交易次数**: 活跃度

## 🔧 API 使用（高级）

### 创建用户
```bash
POST /api/v1/trading/user/create
{
  "user_id": "user123",
  "username": "张三",
  "initial_cash": 100000
}
```

### 买入股票
```bash
POST /api/v1/trading/trade/buy
{
  "user_id": "user123",
  "symbol": "AAPL",
  "shares": 10
}
```

### 卖出股票
```bash
POST /api/v1/trading/trade/sell
{
  "user_id": "user123",
  "symbol": "AAPL",
  "shares": 5
}
```

### 查询投资组合
```bash
GET /api/v1/trading/portfolio/user123
```

### 查询交易历史
```bash
GET /api/v1/trading/history/user123?limit=20
```

### 获取排行榜
```bash
GET /api/v1/trading/leaderboard?limit=10
```

### 获取股票价格
```bash
GET /api/v1/trading/price/AAPL
```

## ⚠️ 注意事项

### 数据说明
- ⚠️ **模拟价格**: 基于真实股价 ±2% 波动
- ⚠️ **不是真实交易**: 使用虚拟资金，无法提现
- ⚠️ **价格延迟**: 模拟系统，非实时行情

### 使用限制
- 单笔交易最少 1 股
- 卖出不能超过持仓数量
- 买入不能超过可用资金
- 交易记录最多显示 50 条

### 账户管理
- 每个浏览器会话自动创建独立账户
- 关闭浏览器后数据保留
- 可以使用"重置账户"功能重新开始

## 🐛 常见问题

### Q1: 为什么买入失败？
**A**: 可能原因：
- 资金不足（检查可用资金）
- 股票代码错误
- 输入数量为 0 或负数

### Q2: 如何查看历史收益？
**A**: 
- 查看"交易记录"表格
- 每笔卖出都显示盈亏
- 总收益看"收益率"卡片

### Q3: 价格为什么一直变化？
**A**: 
- 模拟系统添加 ±2% 随机波动
- 模拟真实市场的动态变化
- 每次查询都会重新计算

### Q4: 可以重置账户吗？
**A**: 
- 目前需要清除浏览器缓存
- 或使用不同浏览器/隐身模式
- 未来版本会添加重置按钮

### Q5: 排行榜如何计算？
**A**: 
- 按**总收益率**排名（不是绝对金额）
- 收益率 = (总资产 - 初始资金) / 初始资金 × 100%
- 实时更新
