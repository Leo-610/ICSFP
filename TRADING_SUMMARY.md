## 🎯 项目目标

为 ICSFP（智能因果股票预测平台）添加一个**趣味性模拟交易模块**，让用户能够：
- 使用虚拟资金进行股票交易
- 验证预测平台的预测准确性
- 学习投资策略和风险管理
- 参与收益排行榜竞争

---

## 📦 已实现功能

### 1. 后端系统 ✅

#### 核心交易引擎 (`api/trading.py`)
- ✅ `TradingSimulator` 类（500+ 行代码）
- ✅ 用户账户管理系统
- ✅ 买入/卖出交易逻辑
- ✅ 持仓管理和成本计算
- ✅ 交易记录存储
- ✅ 排行榜系统
- ✅ JSON 数据持久化

**特色功能**：
- 平均成本自动计算
- 手续费机制（0.1%）
- 盈亏统计（金额和百分比）
- 实时价格模拟（±2% 波动）
- 支持 33 只股票

#### API 路由 (`api/trading_routes.py`)
- ✅ POST `/api/v1/trading/user/create` - 创建用户
- ✅ GET `/api/v1/trading/user/<id>` - 查询用户
- ✅ GET `/api/v1/trading/portfolio/<id>` - 查询持仓
- ✅ POST `/api/v1/trading/trade/buy` - 买入股票
- ✅ POST `/api/v1/trading/trade/sell` - 卖出股票
- ✅ GET `/api/v1/trading/history/<id>` - 交易历史
- ✅ GET `/api/v1/trading/leaderboard` - 排行榜
- ✅ GET `/api/v1/trading/price/<symbol>` - 股价查询
- ✅ POST `/api/v1/trading/user/<id>/reset` - 重置账户

**API 特点**：
- RESTful 设计
- JSON 响应格式
- 完整错误处理
- 日志记录
- 参数验证

## 💡 使用示例

### 场景 1: 新手第一次交易
```
1. 访问 http://127.0.0.1:5000/trading
2. 查看账户: $100,000 初始资金
3. 选择 AAPL，输入 10 股
4. 点击"买入"
5. ✅ 成功！查看持仓显示 10 股 AAPL
```

### 场景 2: 结合预测平台
```
1. 在预测平台查看 MSFT 预测
   → UP 方向，置信度 75%
2. 进入交易平台
3. 买入 MSFT 20 股
4. 等待价格上涨
5. 卖出获利 8.5%
```

### 场景 3: 分散投资组合
```
策略: 投资 5 只不同行业股票
- AAPL (科技)  10股
- JPM  (金融)  15股
- JNJ  (医药)  10股
- XOM  (能源)  20股
- KO   (消费)  30股
结果: 风险分散，收益稳定
```

### 场景 4: 排行榜冲榜
```
目标: 进入前3名 🥇🥈🥉
1. 查看当前排行榜
2. 分析龙头的策略
3. 优化自己的持仓
4. 追求更高收益率
5. 实时监控排名变化
```
