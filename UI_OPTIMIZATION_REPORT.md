## 🔄 智能路由

修改了 `api/app.py` 的 `/health` 路由：

```python
@app.route('/health')
def health_page():
    """智能返回：浏览器→HTML页面，API→JSON"""
    # 如果是浏览器访问，返回HTML页面
    if request.headers.get('Accept') and 'text/html' in request.headers.get('Accept'):
        return app.send_static_file('health.html')
    # 如果是API请求，返回JSON
    return jsonify({
        'status': 'healthy',
        'service': 'ICSFP API',
        'version': '1.0.0'
    })
```

**优势**：
- 浏览器访问 → 显示美观的监控页面
- API请求 → 返回JSON数据
- 向后兼容，不影响现有API调用
